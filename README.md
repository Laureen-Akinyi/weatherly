
Weatherly.

Weatherly react application provides updates for the 
next five days weather data for a city. Weatherly help users get precise weather updates.
Live link [weatherly](https://weatherlyreactapp.netlify.app)

REACT_APP_API_KEY

REACT_APP_URL
